# 营口港码头专业化系统
采用maven、springboot、mybatis、mysql、vue、element-ui开发。


| 文件              | 说明        |
|-----------------|-----------|
| controller      | 接口层       |
| enums           | 返回结果枚举    |
| mapper          | 数据库持久层    |
| pojo            | 数据库实体层    |
| result          | 返回结果类     |
| service         | 业务逻辑层接口   |
| service/impl    | 业务逻辑层接口实现 |
| util            | 工具类       |
| DemoApplication | 程序启动入口    |
| resource        | 资源文件      |
| resource/mapper | sql映射文件   |
| application.yml | 配置文件      |
| pom.xml         | maven依赖   |
